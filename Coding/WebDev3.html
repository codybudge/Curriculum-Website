Web Development 3 - Interactive Webpages

Webpages allow you to add any kind of information that you want, but they can also be used to allow users to interact with them! In this quest you'll create a page that uses the actions from the user to change what the page looks like. Most websites also have more than one page, so you will create multiple pages that can be navigated to and from. Get ready to rock with your interactive webpages!
