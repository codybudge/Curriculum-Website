Comments are the section of code within the "<!--" and "-->" anything between these series of symbols is ignored by the browser window, so you can use them to leave yourself messages, or to label your code.

Make sure to add comments as you code for debugging purposes later. To finish off our <head> section we are going to use the <link> tag to connect to, and use the code written in the css file. This is like a telepathic connection between this file, and the file named in the "href".

href=””  tells our html file where to find our css file.

rel=”” specifies the relationship between our html file and the css file.

../css/PopUpGeekoStyle.css navigates you back one folder then takes you to your css folder and then to your css file. BE SURE to include the ".." before the first slash.
