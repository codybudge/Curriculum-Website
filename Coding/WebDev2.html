Web Development 2 - Intro to IDE's and Stylesheets

How do we type the instructions to send to the computer? Well, we can’t just type it up in an email like we would for anyone else, we must have somewhere to type and create the files. You can use a simple text-editor like Notepad, or you can use a very useful Integrated Development Environment, also called an ‘IDE’. IDE’s provide you with many different tools to assist you in writing, editing, and revising code. For this quest we are going to use an IDE to recreate a famous abstract painting by Piet Mondrian from 1930 called the Composition with Red Blue and Yellow. Creative name huh?

Step 2

CSS

Now it’s time to write the CSS file, CSS stands for Cascading Style Sheets and is used to customize how text from our HTML document displays. We’re only going to write a few lines but you’ll notice a lot of difference from HTML syntax. We enclose our h1 styling with curly brackets “{}” because if we didn’t the computer wouldn’t know what section of the html file to style! Don’t worry about the green squiggly line under h1, that’s just the computer letting us know we aren’t doing anything with it yet.

Now we’re going to start some coding, be careful not to confuse HTML and CSS syntax because they are different. The most noticeable and important would be the “ ; “ and the end of each line, this is how we tell the code we are ending statement, much like how we use periods to end sentences in writing a story. The text-align function is what we can use better position our text instead of defaulting to the left. The color function seems simple enough but then there’s some letters and numbers mixed in, this is called a hexadecimal color code. Every variation of a color you can think of has its very own hexadecimal color code. Now open up your html file and see the changes!
