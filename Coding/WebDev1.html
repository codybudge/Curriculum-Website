Web Development 1 - Intro to Programming

Step 1

GETTING STARTED

What is HTML?

Programming is writing instructions for a computer. Code has different languages, like how humans speak different languages. Web Development uses multiple languages, but starts with HTML. HTML stands for Hypertext Markup Language. In the same way that writing has grammar rules, coding has syntax rules, so everything must be spelled and arranged correctly.

To get started, open your folder in Documents, using the file browser. Then create a new Text Document called "My First Website".

Create a New TXT Document

Go to you folder in the File Browser. Then create a new Text Document called "My First Website".

Step 2

CODING

Writing Your First Code

Double-click to open the new .txt document. The program you're using to add and edit text is called Notepad, a common Text-Editing software that can be used to write in all kinds of coding languages. This empty canvas is where you will make your first coding masterpiece! To begin, type the highlighted text.
The "h1" within the brackets is a special syntax used in HTML, called a tag. Tags are used in pairs, one as an opening (<), and the other as a closing (>). The tags modify the text within them. One way to think about them is to pretend that they are the bread to your code sandwich!

Saving and Changing the File Type

Click File, then Save (or press Ctrl+S). Close the document and go to your folder, where you first created the .txt document.

Left-click on the file to select it, then left-click a second time to edit it's name. Change your extension from .txt to .html. and press Enter - then open it!
Notice how the comment doesn’t show up in our web page, this is a great way to keep track of what we are doing with our code, also notice how our "Hello" within the <h1> tags is bigger. This is all part of the HTML syntax that we can use to make our web pages look better.

STEP 3

Tags and More Tags

Adding Images

Right-click the image to the right and select "Save Image As". Save it as “World.png” into your folder.

The image must be in the same folder as your HTML document, or it won't show up on your website.
Since we can't make edits to your webpage in the browser, you have to use the text editor to code again. Open your "My First Webpage.html" with the notepad by right clicking on it, and choosing Open With → Notepad. Now that we have our file open in our notepad and our web browser we can put them side by side and see our input and output.

More Tags

Next we are going to add a bunch of different tags. Notice how with each <h> tag increasing numbers the size gets smaller and smaller. Also note the <p> paragraph tags can also separate text, however if you want a break in the text you need to add <br/>. Make your own webpage using these tags. Once you’re done typing code in the notepad press “ctrl” and “S” keys to save, then move over to your web browser and press “ctrl” and “R” to reload the page so we can see our changes. Use the image and these new tags to create your own Hello World page!
