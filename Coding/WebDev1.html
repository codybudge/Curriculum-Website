Web Development 1 - Intro to Programming

Step 1

What is Programming?

Programming is writing instructions for a computer. And just like how instructions for people can be written in different languages like English, Francais, Espanol - Instructions for computers can be written in different Programming Languages. There are many different web development languages. Web Development starts with HTML. HTML stands for Hypertext Markup Language and is used to organize text on a web page. In the same way that writing has rules, called grammar, each programming language has its own syntax, or set of rules to follow while programming.

To get started, open your folder in Documents, using the file browser. Then create a new Text Document called "My First Website".

The "h1" within the brackets is a special syntax used in HTML, called a tag. Tags are used in pairs, one as an opening (<), and the other as a closing (>). The tags modify the text within them. One way to think about them is to pretend that they are the bread to your code sandwich!
