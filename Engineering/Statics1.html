Statics 1 - Intro to Building

Step 1

Tools of the Trade

Statics is the study of forces acting on structures that don't move. The Statics quests all center around building great structures to withstand forces.
The building manipulatives that you will use are called Fischertechnik. These are very versatile and can be used and connected in many different ways with their pins and grooves. Notice to the right that there is a Building Block 30. Use the mouse to drag it around and check it out.
We always make sure to use the names of the pieces properly, so that instructions are clear, and instructors can be more helpful.
Rivets are a connection system that allow you to attach Struts and other supports to girders.
To attach a strut to a girder, line up the bores and insert the rivet.
Twist the rivet to lock it in place. Make sure that the rivet is perpendicular to the girder. Rotate the model here to view how the rivet locks to the inside of the girder.

Step 2

Post and Lintel

The fundamental structure you will be building first is the Post and Lintel. This style of structure is used ALL over the world, and has been for CENTURIES! One example of the Post and Lintel is Stonehenge. Lets get started!

1. Start the Posts

Attach the four Building Block 30's to the Base Plate

2. Finish the Posts

Attach a Building Block 15 with 2 Pins to each Building Block 30

3. Start the Lintel

Attach a Building Block 30 to the top of each post. Be sure to line the grooves and pins up so that they are facing the same way on every post

4. Finish the Lintel

Add the last two Building Block 30's

4. Finish the Lintel

Add the last two Building Block 30's
