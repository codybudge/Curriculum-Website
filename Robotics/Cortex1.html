Cortex 1 - Intro to Robotics

Cortex is a user-friendly, visual programming software that is used to program The Brain robotic microcontroller. Cortex and The Brain are compatible with Fischertechnik manipulatives, and can be combined into incredible robotic inventions. Cortex was developed locally as a means to teach programming logic without the need to learn a new language. Robotics is an ever-growing field, with more opportunities appearing every day. So whether it’s a DIY home project, or a multi-million dollar corporate project, robotics is everywhere.

Step 1

BUILD A SIMPLE MOTOR STATION

Gather the Fishertechnik parts you will be using.

Materials List

Building Block 30 (1)
Building Block 15 with 2 Pins (2)
DC Motor (2)
Clip Adapter (2)
Motor Reducing Gearbox (2)
Link 30 (2)
Mounting Plate with Peg 30x45 (1)
Gear Axle T20 (2)

Create the Simple Motor Station

Below is a set of building instructions for the Simple Motor Station. Press the Play icon to see each animated step. Click and drag to turn the model, and use the mouse wheel to zoom in and out. When you finish each step, click the right arrow to move to the next one.

1. Build the Base

Attach the Base Plate and the Building Block 15 with 2 pins to the Building Block 30

2. Attach the motors

Slide each Motor onto a pin of the Building Block 30 with 2 pins.

3. Assemble the Gear Boxes

Insert a Gear Axle into each Gear Box, then attach the Gear Boxes to the Motors so that the Axles are facing outwards

4. Finish the Simple Motor Station

Slide 1 Link 30 into each Clip Adapter, then snap the Adapters onto the Clip Axles

Step 2

GET STARTED WITH CORTEX

Create an account

Open up cortex and click Play. Once open create a new user with your name and pick a color. Note: cortex profiles don’t transfer from computer to computer, so make sure to use the same computer every time!
Once you have your own profile create a new program and get familiar with the command drawers, we won’t be using them all at once but note how each command type has their own color.

Step 4

PROGRAMMING

Creating your first program

Follow along bellow to create your first program. When using the num command the number we put in is a 10th of a second. Meaning 50 represents 5 seconds. Make sure to save your program and check your connect, after that compile and run.

Setting Motor Power

For our second program and a new motor command so our program knows we’re only talking to motor A, then use the set power command and the slider to set motor’s A power to 50%. Next save the program as a new program, “Project2”.
