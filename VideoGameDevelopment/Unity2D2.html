1. Space Shooter Introduction

Great job! Continue your project by following along with the videos below.

Open a new Unity 2D project to get started. To do this, double-click the Unity icon on the desktop, and then click "New". When naming projects, always make sure it's in your folder, and has a name that describes the project.

When your project is loaded, navigate to the Asset store and import the Space Shooter tutorials Asset Package. This contains all of the models and artwork that you will use in this game.

Save your scene in a new folder called "_Scenes". Placing the "_" in front of a folder name makes the folder go to the top of the assets folder, and easier to find. Use this trick to keep your folders organized in the Project window.

Drag the Game window tab next to the Scene window tab. Then change the aspect ratio to 6:9.
