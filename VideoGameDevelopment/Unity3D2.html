UNITY 3D 2 - ISOMETRIC SURVIVAL GAME

Step 1

OPEN A 3D UNITY PROJECT

Open a new Unity 3D project to get started. To do this, double-click the Unity icon on the desktop, and then click "New". When naming projects, always make sure it's in your folder, and has a name that describes the project. In this project, include the Survival Shooter Asset Package by clicking "Add Asset Package" and choosing "Survival Shooter Tutorial".

IF you do not see the "Survival Shooter Tutorial" option, create the project without it, and it can be added later.

This project requires some extra parts that we won't be making ourselves. These assets are available in the Asset Store. When the project loads, watch the instructional below to see how to access the Asset Store, and get these necessary parts into your project.

Open the Asset Store, and search for Survival Shooter. Download and import this into your project.
