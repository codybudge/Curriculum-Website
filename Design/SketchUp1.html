SketchUp 1 - Intro to 3D Modelling

3D modelling is using software to create 3-dimensional images. It can range in uses from creating digital blueprints that are used to autonomously create products, to animating and rendering complete movies. Many different types of 3D modelling exist - the most basic of which is Mesh Modelling. Think of a cube - it has 6 sides. Each of those sides is called a “Face”. With mesh modelling, you can modify those faces, and even add new ones to create an entirely different shape!

Step 1

Opening and Using the Software

Open SketchUp.

First, double-click the SketchUp icon to open the program.

Then click "Choose Template" and choose "Woodworking - Millimeters".

Last, click "Start Using SketchUp".

Let's draw a rectangle

Select the Rectangle tool.

Click on the Origin (the center of the colored lines).

Move the mouse to resize the rectangle.

Click again to create a rectangle.

Create a 3D prism.

Select the Push/Pull tool.

Click on the face of the rectangle.

Move the cursor to pull the surface.

Click again to set the height of the prism.

Moving in 3D spaces is important. You can move with 3 tools: Zoom, Orbit, and Pan.

The Zoom tool moves the model closer or further from your view.

Select the Zoom tool

Click and Hold to zoom in and out

The Orbit tool moves your view around the model (Like the earth orbiting the sun).

Select the Orbit tool

Click and Hold to orbit the model

The Pan tool moves your view from side to side, or up and down. It will not rotate.

Select the Pan tool

Click and Hold to zoom in and out

Step 2

Creating a Chair

Step-by-step 3D chair

SketchUp is generally known as an architectural CAD program. CAD stands for Computer-Aided Design. The tools that you use in a CAD program aid you in drawing your 3D models. Architecture is the designing of buildings. To learn the ropes, we will make a model of a house. So get ready, you "CAD" do it!

Follow along with the slides below. When you complete one step, click the right arrow to continue to the next one!

1. Draw a square and pull it up

A diagonal line will appear when the rectangle is squared

2. Draw squares for the legs

Place the legs on the bottom. Hover over the previous square to make SketchUp "remember" its size

3. Pull the legs

Match the lengths by holding the cursor at the end of the previous leg

4. Make the backrest

Draw a rectangle on top, then pull the backrest up

5. Create the arms

Start with one square, then match the other square with the hovering technique. Pull both to the same length as the front of the chair

6. Make arm supports

Touch the top of the seat when pulling to match the length

Step 3

Creating a House

Model an L-Shaped house

Use your skills to create a model of an L-shaped house. Many projects in SketchUp require a great deal of steps and detail. Be sure to save often, and if you make a mistake, press CTRL+Z to undo it!

1. Draw an L-shaped base

Start at the origin. Use two rectangles, and start the second rectangle on the corner of the first

2. Lift the base together

Erase the line, then use the Push/Pull tool

3. Draw the top of the house

Use the line tool. Hover over the midpoint of the roof's edge to "remember" that distance. Hover over the midpoint of the opposite edge to measure the correct distance. Draw both lines

4. Raise the roof

Hold shift while using the Select tool to select multiple objects. Press the UP ARROW while using the Move tool to lock to the Blue Axis

5. Quickly change tools and view

Press SPACEBAR to choose the Select tool. Press and hold the MIDDLE MOUSE BUTTON to Orbit without selecting the Orbit tool. When released, you will still have the last tool you used active

5. Draw an archway

Use the 2-Point Arc tool. Click once on the top left of the rectangle, once on the top right of the rectangle, then move the cursor up and click to place the height of the arc

6. Finish the entryway

Click the line with the Erase tool. Pull the outer walls forward

7. Make a chimney

Orbit quickly by holding the MIDDLE MOUSE BUTTON, instead of selecting the Orbit tool. Pan quickly by holding SHIFT + MIDDLE MOUSE BUTTON

8. Draw identical windows

Use the Rectangle tool. Hover over the previous windows to use them as reference geometry

9. Copy the rest of the windows

Use the Select tool. Hold LEFT MOUSE BUTTON to drag a selection box. Press CTRL+C to copy the selection. Orbit and press CTRL+V to make a copy of the windows. Click to place the windows

9. Draw the front door and window

For the window, click the Polygon tool. Press 8, then press ENTER to change the polygon to 8 sides

9. Add details

Use the Offset tool. Double click a shape to offest the same distance as the last use of the Offset tool

9. Add textures to the house

Use the Materials Tray to select materials for the house. Click a material to select it. Click a surface on the model to add the material

9. Add textures to the house

Use the Materials Tray to select materials for the house. Click a material to select it. Click a surface on the model to add the material
